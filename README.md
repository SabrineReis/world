🌍 ShareTrip — Landing Page de Viagens
Landing page com tema de viagens, apresentando um planeta Terra girando animado, formulário de busca de roteiros e barra de navegação responsiva. Desenvolvida com HTML e CSS puro.

🚀 Funcionalidades

Planeta animado com rotação contínua (CSS @keyframes)
Formulário de busca com campos de destino, data e número de pessoas
Navegação responsiva com botões de Entrar e Cadastrar
Imagem de fundo em tela cheia com conteúdo sobreposto
Paleta de cores personalizada e integração com Google Fonts


🛠️ Tecnologias Utilizadas

HTML5
CSS3 (Flexbox, Animações, Variáveis CSS)
Font Awesome — ícones
Google Fonts — Open Sans
